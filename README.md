# QR Code Generator (Python + Tkinter + pillow)

A simple QR Code Generator built with Python, using the qrcode library to generate QR data, Pillow to render and save the QR code image, and Tkinter for an easy-to-use graphical interface.
This app lets you generate,  QR codes for any text, URL, or custom data input.
